Vite Permission Error Fix

Quick solution for Vercel deployment issues

sh: line 1: /vercel/path0/node_modules/.bin/vite: Permission denied
Error: Command "npm run build" exited with 126

Solution 1: Add Postinstall Script

Edit your package.json and add a postinstall script:

{
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview",
    "postinstall": "chmod +x node_modules/.bin/*"
  }
}

Solution 2: Modify Build Script

Change your build script to use Node.js directly:

{
  "scripts": {
    "dev": "vite",
    "build": "node node_modules/vite/bin/vite.js build",
    "preview": "vite preview"
  }
}

Important: After making these changes, commit and push to trigger a new Vercel build.
